# Changelog

## 0.16.0

**Professional UI Overhaul**
* Redesigned the entire interface, meticulously optimized for advanced data management and visual clarity.

**Frictionless Workflow Refinements**
* **Dual-Panel Layout**: Replaced the previous right-side drawer with a fully integrated, dual-panel Connection Manager.
* **State Synchronization**: Saved connections now sync instantly and seamlessly between the Main screen and Connection Manager.
* **Context-Aware Editing**: Automatically highlights the selected server and pre-populates connection details directly from the main list.
* **Automated Initialization**: Opening the global manager instantly prepares a blank template, bypassing unnecessary manual clicks.

**Enhanced Connection Manager**
* **Desktop-First Dropdowns**: Overhauled the server type selection menu to achieve pristine text alignment, enhanced readability, and a native desktop feel.

**Optimized Binary Sizes**
* **macOS**: 19.1 MB (`.dmg` installer)
* **Windows**: 12.1 MB (`.zip` installer)
* **Linux**: 10.4 MB (`.AppImage`)

**macOS Under-the-Hood**
* Migrated dependency management from CocoaPods to Swift Package Manager (SPM) for improved build stability.
