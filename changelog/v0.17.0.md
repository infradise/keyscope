# Changelog

## 0.17.0

### Main & Global UI
* Added the following tabs to the **Details view** in the main window’s server list:
  * Command Palette
  * Stats
  * Dashboard
  * Live Watch (temporarily placed here, planned to move to the Key Manager in the side panel)
* **Main UI**
  * Improved internal handling of the delete action in the server list.
  * Added tooltips and text truncation for server names in the server list.
* **Global UI**
  * Refined button style: adjusted rounded rectangle corners to be closer to square for a more stable appearance.

### Enhanced Connection Manager
* Improved Provider Template handling.
* Enhanced internal processing for add/delete operations.
* Enabled smooth auto-scroll to the relevant position when opening the Connection Manager via the **Edit icon** in the server list action buttons.
* Restored smooth auto-scroll to newly added items when creating a connection via the **[+] button** in the Connection Manager.
* **ListTile improvements**: resolved potential warnings about invisible background highlights or ripple effects, reducing unnecessary debug messages.

### Binary Sizes
* **macOS**: 19.1 MB (`.dmg` installer)
* **Windows**: 11.4 MB (`.zip` installer)
* **Linux**: 10.5 MB (`.AppImage`)
