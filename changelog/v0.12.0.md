# Changelog

## 0.12.0
* **Security**
    * Added: **OS Keychain Integration** — Store passwords and connection strings securely using native OS keychain features.
* **Multilingual (i18n)**
  - Removed Simplified Chinese from supported languages

---

**Breaking Changes**  
This version is not compatible with previous versions (e.g., 0.11.0 or earlier) because the Connection Manager now uses the OS keychain. Before using this version, you need to back up your connection information.
