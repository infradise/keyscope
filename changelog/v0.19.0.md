# Changelog

## 0.19.0

### Cloudflare

#### R2 Object Storage
* Added **Cloudflare R2 Storage Integration** (create, delete, and empty buckets; view bucket information; create, upload, download, rename, and delete objects natively).

#### D1 SQLite Database
* Added **Cloudflare D1 Database Integration** (execute SQL queries and view results).
